# 🏭 Kothar Sistema

Sistema de gestão para estúdio de impressão 3D.

## 📌 Funcionalidades

- 📦 Almoxarifado de materiais e manutenção
- ⚙️ Cadastro de equipamentos
- 🧠 Precificação inteligente de produtos
- 🏗️ Controle de produção
- 💰 Controle financeiro
- ♻️ Sistema de reaproveitamento de material

---

## 🧠 Lógica do sistema

### Materiais
- Controle em **gramas**
- Preço baseado em **R$/kg**
- Cálculo automático de custo por grama

### Produção
- Cálculo de custo baseado em:
  - Material
  - Energia (kWh)
  - Tempo de máquina

### Financeiro
- Controle de:
  - Capital em estoque
  - Produção
  - Faturamento
  - Lucro
  - Reaproveitamento
